# Adinkra Worlds V1 — paquet PWA de test

Ce dossier contient une PWA installable après déploiement HTTPS, par exemple via GitHub Pages.

Commandes locales :

```bash
npm install
npm run dev
npm run build
```

Sur téléphone, l’installation se fait depuis l’URL publiée :
- Android/Chrome : menu ⋮ > Installer l’application ou Ajouter à l’écran d’accueil.
- iPhone/Safari : Partager > Sur l’écran d’accueil.

Note : ce paquet est un build de test pour validation d’installation. Le canvas a aussi été corrigé pour le bug du jeu “Conte”.
