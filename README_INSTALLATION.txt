ADINKRA WORLDS — VERSION TEST PWA

Ce paquet correspond à une version test installable de l'application développée dans le canvas.
Ce n'est pas un APK Android natif : c'est une PWA installable depuis le navigateur.

INSTALLATION SUR TÉLÉPHONE ANDROID
1. Décompressez le ZIP.
2. Hébergez le dossier sur un petit serveur HTTPS, par exemple Netlify, Vercel, GitHub Pages ou votre site.
3. Ouvrez l'URL sur Chrome Android.
4. Menu ⋮ > Ajouter à l'écran d'accueil, ou utilisez le bandeau Installer si Chrome le propose.

INSTALLATION SUR IPHONE
1. Hébergez le dossier sur HTTPS.
2. Ouvrez l'URL dans Safari.
3. Partager > Sur l'écran d'accueil.

TEST LOCAL SUR ORDINATEUR
Depuis le dossier :
python3 -m http.server 8080
Puis ouvrir : http://localhost:8080

LIMITES DE CETTE VERSION TEST
- PWA installable, pas APK natif.
- Les XP sont sauvegardés localement dans le navigateur.
- Le scanner et le dessin sont des fonctions de test simplifiées dans ce paquet.
- La V1 complète avec 103 SVG officiels sera une étape séparée.
