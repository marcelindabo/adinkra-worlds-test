# Adinkra Worlds V1 PWA

Cette archive contient une V1 structurée et installable en PWA.

## Contenu
- 103 SVG officiels extraits du zip original dans assets/icons/
- Bibliothèque structurée : adinkra_library_v1.js et src/data/adinkra_library_v1.json
- App PWA : index.html, app.js, styles.css, manifest.webmanifest, service-worker.js

## Installation via GitHub Pages
1. Dézippez cette archive.
2. Envoyez tous les fichiers à la racine du dépôt GitHub Pages.
3. Activez GitHub Pages : Settings > Pages > Deploy from branch > main / root.
4. Ouvrez l’URL sur téléphone.
5. Android/Chrome : menu ⋮ > Ajouter à l’écran d’accueil.

## Statut
V1 technique : les 103 SVG sont disponibles. Les 8 symboles principaux ont du contenu riche, les autres ont du contenu provisoire à compléter.
